Overview
--------

This documentation site is generated with Sphinx. It includes a short getting started guide and API reference pages auto-generated from docstrings.

Build the docs (from repository root):

.. code-block:: sh

   cd docs
   pip install -r requirements-docs.txt
   make html

The built site will appear in `docs/_build/html/`.
